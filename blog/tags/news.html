Open discussionsâ€‹

  • License/Copyright Header and new year - do we have to update all files in the repository?
    No, you don't have to. When you touch a file, update the License/Copyright header like this:

    /*******************************************************************************
     * Copyright (c) 2023, 2024 Contributors to the Eclipse Foundation
    [...]

  • How to contribute to the catena-x.net website
    The catena-x.net website is owned by the Catena-X Association and not maintained by Eclipse Tractus-X. Please get in contact with Stephan Bauer, he will get you in contact with the correct contacts.

  • How to contribute to Eclipse Tractus-X the first time
    All contributions to Eclipse Tractus-X are appreciated, but you need to fulfill some requirements:

    • You need to register an Eclipse Account.
    • You need to sign the Eclipse Contributor Agreement (ECA), otherwise your Pull Request check (for ECA) will fail and not be accepted/merged.
    • Add your GitHub ID to your Eclipse profile (Personal Information â†’ Social Media Links â†’ GitHub Username).
